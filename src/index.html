Курсы с горячими скидками
Quo-Lab® HbA1c Analyzer
Quo-Lab ® HbA1c - это полуавтоматический настольный анализатор в месте оказания медицинской помощи для измерения гликированного гемоглобина (HbA1c).
Quo-Test® HbA1c Analyzer
Quo-Test ® представляет собой полностью автоматизированный рабочий стол пункт-уход анализатор предназначен для легкого и надежного измерения HbA1c для контроля и управления диабетом
Hemo Control
Hemo Control от EKF Diagnostics - это анализатор гемоглобина на месте, предназначенный для получения количественных результатов лабораторного качества за 25 секунд.
Biosen C-Line
Biosen может тестировать кровь, плазму или сыворотку для получения значений глюкозы и лактата с превосходной точностью (менее 2% CV) в широком диапазоне измерений.
Lactate Scout 4
Простой, точный и надежный анализатор лактата для мониторинга спортивных результатов.
